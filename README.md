# Merchant Aggregator

个人向桌面聚合工具：

- **PriceAI**：只用于同步商家列表
- **价格**：以发卡网（当前 ldxp）店内商品为准
- 同步方式：**一键初始化 / 增量 / 单店 / 多选 / 强制全量**
- 本地 SQLite 搜索比价（搜索永不联网）

## 开发

```bash
pnpm install
pnpm dev
pnpm test
pnpm typecheck
```

## 使用

1. 首页「一键初始化」：同步商家列表 + 报价数 Top 50 店铺商品，完成即可搜货
2. `⌘/Ctrl+K` 搜索本地店内商品：
   - 结果内直接标注「最低」价与差价、价格新鲜度（超龄可单店刷新）
   - 无数据/无命中时，按关键词推荐待同步的候选店，一键补数据后自动重搜
   - `↑↓` 选行、`Enter` 打开源站、「加载更多」翻页
3. 增量刷新：同步中心「增量同步 ldxp」自动跳过新鲜期内成功的店（新鲜期见设置，默认 24h）；
   「强制全量」忽略新鲜期；部分失败的任务可展开明细并「重试失败的店」
4. 收藏 = 盯价清单：显示当前价/库存/更新时间，「刷新收藏的店」只重刮相关店铺
