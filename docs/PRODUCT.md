## 是什么

个人向桌面比价聚合工具。从 **PriceAI / NodeBits** 同步商家主档（双列表源，入库后同一店铺一行），平台识别与店内商品深刮为 **全局策略**（与列表来源无关：`identifyShopPlatform` + registry，当前 **ldxp** / **catfk** / 独角 / 异次元等）。本地 **SQLite** 完成跨店搜索、比价、收藏与跳转源站。搜索路径零网络请求；所有联网同步都由用户显式发起。

## 用户与场景

单人使用者（开发者本人）。场景：采购 AI 账号 / 订阅 / 卡密类商品时快速比价。桌面环境、长时间读表格。Hero 用例：**输入关键词 → 跨店看到匹配商品 → 认出最低价 → 打开源站购买**。

## 核心面

- **搜索（首页）**：全局搜索 + 规格 chips（含否定排除，如「非 Plus」）+ 店铺 facet + 分页 + 比价抽屉；冷启动引导并入空态；路由 keep-alive 保留筛选状态。
- **商家**：列表筛选（含 shop 平台、健康度、可刮店）+ 详情主从布局，按店同步商品。
- **收藏与最近**：个人记忆层；可只刷新相关店。
- **同步中心**：初始化 / 增量 / 全量 / 单店与多选；任务历史筛选与分页、详情（错误明细）、删除/清空、重试失败店。
- **设置**：合规控制（限速 / 新鲜期 / UA / 外链策略）。

## 语气

工具风：简短、中性、可执行。不营销、不拟人、不用感叹号。数据永远标注来源与新鲜度。
